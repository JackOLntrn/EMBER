Detailed Description

A class to add basic functions for DRV8871 Motor Driver.

This class enables drive, coast, and brake functions for a DRV8871 motor driver. It uses PWM for speed control and a digital pin for direction control. This class uses 8 bit PWM resolution, so speed values should be integers from 0 to 255.

Template Parameters
in1PWM output pin (uses analogWrite)
in2Direction pin (uses digitalWrite)
